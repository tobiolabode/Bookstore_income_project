## Installation

Use Git Clone to install project

```bash
git clone https://github.com/tobiolabode/Bookstore_income_project.git
```

## Usage
To run the data analysis:

```python

python main.py
```

If you want to use the data scraper functions:
```python

python scraper.py
python borough.scraper.py

```
